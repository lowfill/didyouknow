version 1.0b

This is my first plugin for elgg. It's not really complete yet, but it works fine and might as well share my progress.

It is a really simple plugin. It is a sidebar module that has little pieces of information for new users. The idea came from the File Tools plugin. But that was only for that plugin, and it would be better if it could be for any plugin. So far, this works with the Groups and Event Calendar plugins.

The site I work on is English/French so it will be good for multiple languages.

Thanks to Matt Beckett for helping me.

To customize the tips that show up, check out 'languages/en.php' for instructions.
